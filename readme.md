# Project Statup

- Check if node and npm is installed:
  `node -v`
  `npm-v`

- Install Gulp CLI globally:
  `npm install gulp-cli -g`

- Install the dependencies:
  `npm install`
  _(inside the project folder)_

- run `gulp watch` to run a basic routine configured on gulpfile.

# Minify images

- run gulp task:
  `gulp minify`
